# 五分钟汇报讲稿

各位老师、同学大家好。我们这次项目的主题是：面向高维混沌系统长期推演的物理残差混合深度学习方法研究。我们的实验对象主要包括 Lorenz-63 和 Lorenz-96，其中 Lorenz-63 用来说明问题背景，而真正的核心是 Lorenz-96 10 维系统上的长期多步预测。

首先，我们想强调一个关键点：混沌系统预测的难点，不在于 one-step prediction，而在于 recursive rollout。单步预测时，模型每次输入的都是来自真实轨迹的历史窗口，所以测试分布和训练分布比较接近；但在长期推演时，模型后续的输入会不断包含它自己前面预测出来的状态。这样一来，一个很小的误差会被混沌动力学不断放大，最终轨迹会逐渐偏离真实吸引子附近。

Lorenz-63 的实验正好说明了这一点。我们测试了线性回归、随机森林和 MLP，它们在短期预测上都能学到一定的局部映射，尤其是随机森林和 MLP 的短期 RMSE 比较低。但是一旦进行 500-step rollout，误差都会明显累积。这说明短期拟合能力并不能代表长期推演稳定性。因此后面的工作重点转向 Lorenz-96 10D。

Lorenz-96 是一个更高维、更强耦合的混沌系统。它的每个变量都和邻近变量有关，并且采用周期边界条件。我们设置维度 N 等于 10，真实外力参数 F 为 8.0。这个任务中，我们不只是预测下一步，而是希望模型能够进行 500 步的长期 rollout。因此，我们用历史窗口作为输入，用累计 RMSE 来评价长期推演误差。

接下来介绍模型设计。我们的思路不是一开始就直接做 Ultimate Hybrid，而是逐步分析每个模块为什么有用。第一步是从 Baseline LSTM 改成 Residual LSTM。因为连续动力系统中，相邻状态之间的变化通常比绝对状态更容易学习，所以模型不直接预测下一时刻状态，而是预测从当前状态到下一状态的增量。这样模型更像一个数据驱动的局部积分器。同时，我们加入 LayerNorm 和 gradient clipping 来提高训练稳定性。

第二步是 Scheduled Sampling。普通训练时，模型总是看到真实历史；但 rollout 时，它会看到自己的预测结果，这会造成训练和测试之间的分布偏移。Scheduled Sampling 的作用就是在训练时按一定概率使用模型自己的预测状态，让训练过程更接近真正的长期推演。不过我们的结果也说明，它不是一定带来收益，衰减策略和多步展开方式非常关键。

第三步是 PINN 物理约束。我们根据 Lorenz-96 方程构造物理损失，让模型预测出来的一步变化率尽量和真实动力学方程给出的导数一致。这样做的意义是：即使模型短期误差很小，也不能让它走向明显不符合物理规律的方向。

第四步是引入 Transformer。LSTM 会把历史窗口压缩进一个隐藏状态，而 Transformer 可以通过 self-attention 显式建模窗口内不同时间步之间的关系。因此它更适合捕捉高维相空间中局部运动趋势。

最后，我们设计了 Ultimate Hybrid。它的核心公式是：预测值等于一个不完美物理求解器的结果，再加上 Transformer 学到的残差。这里的不完美物理求解器使用 RK4，但参数故意设为 F 等于 8.5，而真实系统是 8.0。也就是说，物理模型提供大致正确的演化方向，神经网络负责修正参数偏差和模型误差。最终架构同时结合了不完美物理基底、Transformer 残差、PINN 物理约束和 5 步 refined scheduled sampling。

从实验结果看，有两个现象比较重要。第一，单步指标更好不一定代表 rollout 更好。例如 Phase 2 的 one-step RMSE 比 Phase 1 更低，但 500-step RMSE 反而更高。这说明长期稳定性必须单独评价，不能只看 one-step RMSE。第二，历史窗口也不是越长越好。窗口长度为 50 时，500-step RMSE 明显恶化，说明过长历史会引入冗余甚至混沌噪声，使外推更不稳定。

在高级模型对比中，Baseline LSTM 的 500-step RMSE 是 148.79，误差非常大；加入 PINN 后降到 20.18；Refined Scheduled Sampling 和 Transformer 分别达到 16.79 和 16.34；Hybrid LSTM 达到 15.84；最终 Ultimate Hybrid 达到 12.80，是所有模型中最好的。

这说明 Ultimate Hybrid 的优势来自几个方面的协同：物理求解器降低长期漂移，残差学习降低神经网络学习难度，Transformer 利用历史窗口信息，PINN 保持动力学一致性，而 Scheduled Sampling 让训练阶段更贴近 rollout 场景。

当然，这个项目还有局限。比如我们目前只测试了 N 等于 10、F 等于 8.0 的 Lorenz-96，没有覆盖更多维度和参数；训练轮数也统一设置为 20，还没有做大规模超参数搜索；结果主要来自单随机种子，后续还需要补充均值和方差。

最后总结一下：我们的主要结论是，高维混沌系统的核心评价应该是长期 rollout，而不是单步预测；单纯的数据驱动模型容易在长期推演中发散；更有效的路线是以物理模型作为基底，让深度学习模型修正残差。最终，Ultimate Hybrid 在 Lorenz-96 10D 的 500-step 推演中取得了最好的累计 RMSE。我的汇报到这里，谢谢大家。
